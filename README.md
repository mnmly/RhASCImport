## ASC Import Plugin for Rhino

### Installation
Place files into `~/Library/Application Support/McNeel/Rhinoceros/MacPlugIns/`

### Usage
`File` → `Import…` → `Options` → `Esri ASCII raster format (asc)` → `Open`
